Standard Header Files of the lcc-win32 System

The compiler comes with a complete set of header files for the standard ANSI headers and the windows header files. In the table below, the header files have been classified as either :

·        Windows related headers (WIN32)

·        Standard. This files are part of the language as specified by the C99 standard.  (STD)

·        lcc-win32 specific files (LCC)
