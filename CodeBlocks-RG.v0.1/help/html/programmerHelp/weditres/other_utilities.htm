Other utilities

Wedit has many other built-in utilities.

  • A profiler to find out where your program spends its time
  • A binary editor
  • A software metrics module
  • A utility to insert character strings into the executable for identification purposes.
  • Slicing of program text to view all modifications to an indentifier
  • Generation of a string table
  • A "Diff" utility
  • A function "tree" utility
  • A cross referencer for object files
  • A prototype writer
  • Statistics of the executable (size, ect)
  • Generation of the documentation in .hlp format
  • Application generator (wizard)
  • Printing
