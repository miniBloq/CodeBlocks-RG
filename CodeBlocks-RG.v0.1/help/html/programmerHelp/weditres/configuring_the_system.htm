Configuring the system

The configuration of the whole system is done using the property sheet page that appears when you select "Project" then "Configuration" menu option.

The configuration is organized in several pages.

  • General. Options related to the editor (Tab size, etc) and to the current path
  • Compiler. Compilation options
  • Linker. About the linker (output, etc)
  • Source control. To activate/desactivate source versioning
  • Project paths. Shows the different paths to make, etc.
  • Debugger. Name of the executable and arguments to give it when started
  • Workspace: Fonts, colors, auto compile options.
  • Macros. Defines a macro
  • Language. Maintains the keyword list
  • Utilities. Adds or removes a named utility program.
  • System info. Shows different system parameters
