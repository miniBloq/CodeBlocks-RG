Language keywords

Here you can add, delete or change a keyword that will be highlited in the editor. Obviously the editor can't know all the keywords you would like to have emphasized, and starts with a default list for the C language. Using this tab you can change those keywords

The button "Erase" is enabled when you have selected something in the list of existing keywords; The button "Add" is enabled when you have typed something in the "New keyword" edit field.
